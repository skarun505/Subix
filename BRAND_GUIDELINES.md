# Subix Brand Guidelines

## 1. Color Palette

### Primary Colors
These are the core colors of the Subix brand, used for backgrounds, text, and primary actions.

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Ultra Dark (Background)** | `#050505` | Main page background |
| **Surface** | `#111111` | Card backgrounds, sections |
| **Surface Highlight** | `#1a1a1a` | Hover states, distinct areas |
| **Border** | `#333333` | Borders, dividers |
| **Text Main** | `#ffffff` | Headings, primary text |
| **Text Muted** | `#888888` | Body text, subheadings |

### Accent Colors (Acid & Neon)
Used for highlights, buttons, accents, and gradients to create the "Cyber/Tech" feel.

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Lime** | `#ccff00` | CTAs, Highlights, Logos |
| **Primary Hover** | `#b3e600` | Button hover states |
| **Accent Purple** | `#9d4edd` | Gradients, secondary accents |
| **Accent Pink** | `#ff007f` | Gradients, emotional highlights |
| **Accent Cyan** | `#00f3ff` | Tech elements, data visualization |

### Gradients
*   **Text Gradient:** `linear-gradient(90deg, #ffffff, #888888)`

---

## 2. Typography

### Heading Font
**Space Grotesk**
*   **Usage:** Headings (`h1`, `h2`, `h3`, `h4`, `h5`, `h6`), Logo, Buttons
*   **Weights:** 300, 400, 500, 600, 700
*   **Letter Spacing:** `-0.03em` for headings, `-1px` for logo

### Body Font
**Inter**
*   **Usage:** Body text (`p`), Links, Navigation, Small UI elements
*   **Weights:** 300, 400, 500, 600
*   **Line Height:** `1.6`

---

## 3. UI Elements

### Buttons
*   **Primary Button:**
    *   Background: `#ffffff` (Text Main)
    *   Text: `#000000`
    *   Hover Background: `#ccff00` (Primary Lime)
    *   Border Radius: `8px`
    *   Font: Space Grotesk, Bold

*   **Secondary Button:**
    *   Background: Transparent
    *   Border: `1px solid #333333`
    *   Text: `#ffffff`
    *   Hover Border: `#ffffff`
    *   Border Radius: `8px`

### Cards
*   **Background:** `#111111` (Surface)
*   **Border:** `1px solid #333333`
*   **Border Radius:** `16px`
*   **Hover Effect:** Border color changes to `#888888`, `transform: translateY(-5px)`

### Navbar
*   **Background:** `rgba(17, 17, 17, 0.8)` with Backdrop Blur `20px`
*   **Border:** `1px solid #333333`
*   **Border Radius:** `100px`

---

## 4. Visual Effects

*   **Noise Overlay:** A subtle SVG fractal noise overlay is applied to the `body` with `opacity: 0.05` to give a texture.
*   **Scroll Review:** Elements fade in and move up (`translateY(50px)`) upon scrolling into view.
