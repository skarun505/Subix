# Subix - Automated Revenue Systems

Subix is a technology-driven growth company helping startups and businesses build automated revenue systems through SaaS products and performance-focused digital services.

## Project Structure

- `index.html`: Main landing page structure and content.
- `style.css`: Premium dark-mode styling with glassmorphism and animations.
- `script.js`: Interactive elements (mobile menu, smooth scroll, fade-in animations).

## Features

- **Premium Design**: Modern dark theme with vibrant gradients and glass effects.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Smooth fade-in effects on scroll and floating background elements.
- **Sections**:
    - Hero with CTA
    - Services (Revenue, Marketing, SaaS, Automation)
    - Comparison (Traditional vs Subix)
    - Platform (Subix LeadOS)
    - Engagement Models
    - Why Choose Us
    - Contact CTA

## How to Run

Simply open `index.html` in any modern web browser.

No build step required. This is a static site built with vanilla HTML, CSS, and JS for maximum performance and simplicity.

## Assets

Fonts are loaded from Google Fonts (Outfit). Icons are loaded from FontAwesome CDN. Ensure you have an internet connection to view them correctly.
